\## Project Overview

This project focuses on performing statistical analysis on the Yellow Trip dataset to uncover meaningful insights related to trip patterns, fare distribution, and customer payment behavior.



The analysis includes data cleaning, exploratory data analysis (EDA), and hypothesis testing using Python. A detailed report has also been created to summarize findings and insights.







\## Objectives

\- Analyze trip and fare patterns  

\- Perform descriptive statistical analysis  

\- Identify relationships between variables  

\- Apply hypothesis testing  

\- Generate actionable insights
