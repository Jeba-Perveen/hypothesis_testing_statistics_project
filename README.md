##  Project Overview
This project focuses on performing statistical analysis on the Yellow Trip dataset to uncover meaningful insights related to trip patterns, fare distribution, and customer payment behavior.

The analysis includes data cleaning, exploratory data analysis (EDA), and hypothesis testing using Python. A detailed report has also been created to summarize findings and insights.

---

##  Objectives
- Analyze trip and fare patterns  
- Perform descriptive statistical analysis  
- Identify relationships between variables  
- Apply hypothesis testing  
- Generate actionable insights  

---

##  Dataset
- **Dataset:** Yellow Trip Data (January 2020)  
- **Dataset File:** `yellow_tripdata_2020-01(1).csv`  
- **Dataset Source:** https://www.kaggle.com/datasets/gauravpathak1789/yellow-tripdata-2020-01  

### Contains:
- Trip distance  
- Fare amount  
- Payment type  
- Pickup and drop-off details

## Analysis Performed
### Data Cleaning
- Removed missing values
- Fixed data inconsistencies
- Converted data types
  
### Descriptive Analysis
- Calculated mean, median, and standard deviation
- Analyzed distribution of fare and trip distance

### Exploratory Data Analysis (EDA)
- Visualized trends using graphs
- Compared different payment types

### Hypothesis Testing
- Conducted T-test
- Evaluated relationship between payment type and fare amount

### Key Insights
- Fare distribution varies significantly across trips
- Payment type influences fare amount
- Identified patterns in customer behavior
